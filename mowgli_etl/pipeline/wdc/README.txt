This folder contains the following files:

File name:   offers_corpus_english_v2_1000.jsonl
Description: First 1000 entries of Web Data Commons English offers corpus.

File name:   offers_corpus_english_v2_random_100.jsonl
Description: Random sample of 100 products from above file.

File name:   offers_corpus_english_v2_random_sample.py
Description: Generates random sample of 100 products.

File name:   offers_corpus_english_v2_parse.py
Description: Python routines to extract dimension information using regular expressions
             and product name using spaCy. Currently, it reads in the random 100 data
             source, but the file name can be changed to a different source if desired.
